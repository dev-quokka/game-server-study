# 스마트 포인터

## unique_ptr

유니크 포인터는 해당 객체를 가리키는 유일한 포인터임을 보장해주는 스마트 포인터입니다.  
unique_ptr을 사용할 경우 일반적인 복사 생성자를 지원하지 않으며, 오직 move 함수를 통한 전달만 가능합니다.
스택 영역을 벗어나는 순간 가리키는 객체의 할당을 자동으로 해제합니다.

일반 포인터라면 해당 객체를 가리키는 다른 포인터들이 존재할 수 있기 때문에 자동으로 해제하면 안되지만,  
유니크 포인터는 스택 영역을 벗어나는 순간 해당 객체를 가리키는 포인터가 존재할 수 없게 되므로  
자동으로 해제하는 데 문제가 없습니다.

따라서 유니크 포인터로 가리키는 객체는 delete 를 통해 수동으로 해제할 수 없습니다.  
다른 스마트 포인터들도 동일합니다.
